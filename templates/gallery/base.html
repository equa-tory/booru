search syntax

tag1 tag2 — posts that have both tags (AND).
( tag1 ~ tag2 ) — posts that have tag1 or tag2. The spaces around the braces and the ~ matter, because some tags end in braces.
-tag1 — posts that do not have tag1.
night~ — fuzzy match: also returns near-spelled tags like night, fight, bright.
ta*1 — wildcard: tags starting with “ta” and ending “1”. * matches anything.
file:cat — posts whose file name contains “cat”.
folder:trip — posts whose folder path contains “trip”.
Combine freely, e.g.  1girl ( red_hair ~ blue_hair ) -monochrome file:2024
